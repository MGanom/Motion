# Motion

## 소개
워크스페이스 Notion의 기능들을 참고하여 To do, 비디오, 이미지, 그리고 노트를 추가하고 관리하는 기능을 구현한 사이트이다.

## 진행기간
* 1차완성: 211102 ~ 211110
* 디자인 및 코드 정리:

## 사용 기술
HTML, JavaScript, TypeScript

## 기능 설명
